# 🔥 SpeedCool: Libera Todo el Potencial de Tu Android

> **SpeedCool** es el módulo Magisk definitivo diseñado para transformar tu dispositivo Android. Di adiós al lag, el sobrecalentamiento y la lentitud. Con optimizaciones inteligentes, tu teléfono funcionará más rápido, fluido y mantendrá temperaturas óptimas para cualquier tarea, desde juegos intensos hasta el uso diario.

## ✨ Características Principales

SpeedCool va más allá de la optimización básica, ofreciendo un conjunto completo de mejoras:

*   **Rendimiento Mejorado:** Aumenta el rendimiento general y la experiencia de juego, asegurando un sistema fluido y receptivo.
*   **Control Térmico Inteligente:** Reduce significativamente el sobrecalentamiento de la CPU y GPU, prolongando la vida útil de tu hardware y manteniendo el máximo rendimiento.
*   **Gestión Eficiente de RAM:** Libera RAM para un sistema más ligero, receptivo y estable.
*   **Sistema Adaptativo:** Optimizaciones que se ajustan dinámicamente a tus patrones de uso, garantizando un rendimiento óptimo sin comprometer la duración de la batería.
*   **Menú Interactivo:** Control total a través de Termux con un menú fácil de usar para personalizar la configuración y los perfiles.
*   **Amplia Compatibilidad:** Funciona en una amplia gama de dispositivos y chipsets, incluyendo **MediaTek, Tensor, Kirin, Qualcomm, Exynos**, y más.
*   **Perfiles Automáticos Basados en Aplicaciones (v2.2+):** Detecta las aplicaciones en primer plano y cambia automáticamente los perfiles de rendimiento según tus reglas definidas [1].
*   **Actualización Automática Completa a través de GitHub (v2.2+):** SpeedCool busca actualizaciones, descarga el `.zip` y lo instala automáticamente [1].
*   **Copia de Seguridad e Importación de Perfiles (v2.2+):** Guarda y restaura tu configuración fácilmente directamente en Termux [1].
*   **Sistema de Recuperación de Bootloop (v2.2+):** Detecta reinicios repetidos y restaura automáticamente un perfil "eco" seguro, previniendo la inestabilidad del sistema [1].
*   **Diseño de Script Modular (v2.2+):** Scripts separados para rendimiento, enfriamiento, aprendizaje, monitoreo de aplicaciones, actualizaciones y copias de seguridad, permitiendo un funcionamiento y reinicio independientes [1].

## 🏗️ Cómo Funciona

SpeedCool utiliza scripts avanzados que optimizan en tiempo real:

*   **CPU y GPU:** Ajustes precisos para el rendimiento y la temperatura.
*   **E/S (Entrada/Salida):** Acelera la lectura y escritura de datos.
*   **RAM:** Gestión inteligente para evitar cuellos de botella.
*   **Servicios en Segundo Plano:** Optimiza los procesos para conservar recursos.

**Todo esto ocurre automáticamente al inicio, sin una interfaz gráfica para mantener el módulo ligero y eficiente.** Para el control manual, utiliza el menú de Termux.

## 📦 Instalación

Sigue estos sencillos pasos para instalar SpeedCool:

1.  Abre **Magisk Manager**.
2.  Ve a la pestaña **Módulos**.
3.  Toca en **Instalar desde el almacenamiento**.
4.  Selecciona el archivo `SpeedCool_Vx.x.zip` (asegúrate de descargar la versión más reciente).
5.  **Reinicia tu dispositivo** para aplicar las optimizaciones.

### ➕ Acceso al Menú de Control (a través de Termux)

Para acceder a la configuración avanzada y los perfiles de SpeedCool:

1.  Abre **Termux**.
2.  Escribe `su` y concede permiso de superusuario.
3.  Ejecuta el comando:

    ```bash
    sh /data/adb/modules/speedcool/scripts/menu.sh
    ```

---

⚠️ **Advertencias Importantes**

*   💾 **Siempre haz una copia de seguridad** de tu sistema antes de instalar cualquier módulo Magisk.
*   🚫 **Evita Conflictos:** No uses SpeedCool junto con otros módulos de optimización de rendimiento (como FDE.AI o Aionix), ya que pueden ocurrir inestabilidades. **Recomendamos usar solo SpeedCool para prevenir problemas.**
*   ✅ **Desinstalación Fácil:** Puedes desinstalar SpeedCool en cualquier momento a través de Magisk Manager.
*   ⚠️ **Uso bajo tu propio riesgo:** Aunque probado y funcional, el comportamiento puede variar ligeramente según el dispositivo y la ROM.
