## ✨ Основные Возможности

SpeedCool выходит за рамки базовой оптимизации, предлагая полный набор улучшений:

*   **Повышенная Производительность:** Увеличивает общую производительность и игровой опыт, обеспечивая плавную и отзывчивую систему.
*   **Интеллектуальный Термоконтроль:** Значительно снижает перегрев CPU и GPU, продлевая срок службы вашего оборудования и поддерживая максимальную производительность.
*   **Эффективное Управление ОЗУ:** Освобождает оперативную память для более легкой, отзывчивой и стабильной системы.
*   **Адаптивная Система:** Оптимизации, которые динамически подстраиваются под ваши шаблоны использования, гарантируя оптимальную производительность без ущерба для времени автономной работы.
*   **Интерактивное Меню:** Полный контроль через Termux с удобным меню для настройки параметров и профилей.
*   **Широкая Совместимость:** Работает на широком спектре устройств и чипсетов, включая **MediaTek, Tensor, Kirin, Qualcomm, Exynos** и другие.
*   **Автоматические Профили на Основе Приложений (v2.2+):** Обнаруживает приложения на переднем плане и автоматически переключает профили производительности на основе ваших определенных правил [1].
*   **Полное Автообновление через GitHub (v2.2+):** SpeedCool проверяет наличие обновлений, загружает `.zip` и устанавливает его автоматически [1].
*   **Резервное Копирование и Импорт Профилей (v2.2+):** Легко сохраняйте и восстанавливайте свои настройки прямо в Termux [1].
*   **Система Восстановления Загрузочной Петли (v2.2+):** Обнаруживает повторяющиеся перезагрузки и автоматически восстанавливает безопасный "эко"-профиль, предотвращая нестабильность системы [1].
*   **Модульный Дизайн Скриптов (v2.2+):** Отдельные скрипты для производительности, охлаждения, обучения, мониторинга приложений, обновлений и резервного копирования, позволяющие независимую работу и перезапуск [1].

## 🏗️ Как Это Работает

SpeedCool использует передовые скрипты, которые оптимизируют в реальном времени:

*   **CPU и GPU:** Тонкая настройка для производительности и температуры.
*   **Ввод/Вывод (I/O):** Ускоряет чтение и запись данных.
*   **ОЗУ:** Интеллектуальное управление для предотвращения узких мест.
*   **Фоновые Службы:** Оптимизирует процессы для экономии ресурсов.

**Все это происходит автоматически при запуске, без графического интерфейса, чтобы модуль оставался легким и эффективным.** Для ручного управления используйте меню Termux.

## 📦 Установка

Выполните следующие простые шаги для установки SpeedCool:

1.  Откройте **Magisk Manager**.
2.  Перейдите на вкладку **Модули**.
3.  Нажмите **Установить из хранилища**.
4.  Выберите файл `SpeedCool_Vx.x.zip` (убедитесь, что вы загрузили последнюю версию).
5.  **Перезагрузите устройство**, чтобы применить оптимизации.

### ➕ Доступ к Меню Управления (через Termux)

Чтобы получить доступ к расширенным настройкам и профилям SpeedCool:

1.  Откройте **Termux**.
2.  Введите `su` и предоставьте разрешение суперпользователя.
3.  Выполните команду:

    ```bash
    sh /data/adb/modules/speedcool/scripts/menu.sh
    ```

⚠️ **Важные Предупреждения**

*   💾 **Всегда делайте резервную копию** вашей системы перед установкой любого модуля Magisk.
*   🚫 **Избегайте Конфликтов:** Не используйте SpeedCool вместе с другими модулями оптимизации производительности (например, FDE.AI или Aionix), так как могут возникнуть нестабильности. **Мы рекомендуем использовать только SpeedCool для предотвращения проблем.**
*   ✅ **Простое Удаление:** Вы можете удалить SpeedCool в любое время через Magisk Manager.
*   ⚠️ **Используйте на Свой Страх и Риск:** Хотя протестировано и функционально, поведение может незначительно отличаться в зависимости от устройства и ROM.
